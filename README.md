# Pokedex-
Criação de uma Pokedéx consumindo o PokeAPI com JS 
Link do projeto: 
https://mateus-cypriano.github.io/Pokedex-/

<img width="1654" height="1009" alt="image" src="https://github.com/user-attachments/assets/74a224ef-2302-44a4-9062-add34f712aca" />

Utilizando apenas JS para consumir dados da API PokeAPI e manipular os objetos da pagina para renderizar cada pokémon de acordo com a pesquisa(captura das teclas) do usuário seja por ID do Pokémon ou pelo nome.

Só pela foto de perfil já da para ver que sempre fui fã de Pokémon e que esse foi um dos projetos que me inspiraram a estudar programação na faculdade e finalmente coloquei em prática para treinar o consumo de API's e a manipulação de objetos com JS puro.
